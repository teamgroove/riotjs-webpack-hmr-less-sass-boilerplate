# riotjs webpack-hmr-sass boilerplate
* HMR (Hot module reloading) activated

## install
```bash
npm install
```

## development

```bash
npm run dev
```
or

```bash
./node_modules/.bin/webpack-dev-server
```

and
- open http://localhost:8080/

##HMR
Try changing a .tag-file and see the HMR-feature at work. Profit!
